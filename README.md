# Direct Debit Analytics

A collection of product, revenue and operational analytics models built for a Direct Debit platform.

## Business Context

The core business entity is:

User × Merchant × Bank = Mandate

The analyses focus on the complete lifecycle of mandates, transactions, merchants and banks.

---

## Analytics Domains

### Funnel Analytics

* Acquisition Funnel
* Activation Funnel

### Revenue Analytics

* New vs Existing Users
* Revenue Cohort Analysis

### User Behavior Analytics

* Duplicate Active Mandates
* Multi Account Users
* Cross Merchant Usage

### Growth Analytics

* Merchant Growth Monitoring
* Contract Growth Monitoring

### Operational Analytics

* Bank Performance
* Bank Response Time Monitoring
* Hourly Transaction Analysis

---

## Key Business Questions

### Acquisition

How efficiently are users converted from OAuth initiation to first successful debit?

### Activation

How many mandates generate real value after creation?

### Revenue

How much revenue comes from newly acquired users versus existing users?

### User Behavior

Do users create multiple mandates?
Do they use multiple bank accounts?
Do they transact across merchants?

### Growth

Which merchants are growing or declining in mandate creation?

### Operations

Which banks have the highest latency and error rates?
